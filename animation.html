Strong Typing

Dart uses a sound type system, which is statically typed. This means types are checked at compile-time, helping to catch errors early. • Type Inference: While it is strongly typed, Dart`s type inference reduces the verbosity of the code by inferring types when they are not explicitly stated. • Null Safety: Dart ensures that values cannot be null unless explicitly allowed, preventing many common runtime errors. Dart Map
Using the Map Constructor

You can also create a Map using its constructor. This approach is more flexible and allows for the creation of an empty Map that you can populate later. Example
var fruits = Map();
void main() {
fruits ['apple'] = 1;
fruits['banana'] = 2;
fruits['orange'] = 3;
print(fruits);
}
// Output: {apple: 1, banana: 2

Using a Map Literal

This is the most straightforward way to create a Map in Dart. It uses a syntax similar to JSON objects in JavaScript. Example
var fruits = {
'apple': 1,
'banana': 2,
'orange': 3,
};
void main() {
print(fruits);
}
// Output: {apple: 1, banana: 2 }
